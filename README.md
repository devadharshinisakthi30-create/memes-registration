# 🎭 Memes Event Registration

Simple Flask app for memes event registration with QR code access.

## ✨ Features

- 📝 Student registration form
- 🎫 Auto enroll numbers (24HME001, 24HME002...)
- 👥 Horizontal table view of registered students
- 📊 Total count display
- 📱 QR code for mobile access
- 🎨 Modern gradient design

## 🚀 Deploy to Render

1. Push to GitHub
2. Connect to Render
3. Deploy automatically

## 🔧 Environment Variables

Add in Render dashboard:

```
SECRET_KEY=your-random-secret-key
DEPLOYED_URL=https://your-app.onrender.com
```
